# Agent Team: AI Skills System

## Team Composition

| Agent | Role | Expertise |
|-------|------|-----------|
| **skills-architect** | Lead | Designing skill structures, YAML schemas, and AI index logic. |
| **wordpress-integrator** | Dev | Enhancing the Python sync script and WordPress REST API calls. |
| **system-writer** | Writer | Authoring the procedural content for core Geometry OS skills. |
| **test-engineer** | QA | Verifying the end-to-end sync workflow and AI-loadable index. |

## Mission
To implement a robust, dual-purpose AI Skills System for Geometry OS that bridges the gap between AI procedural execution and human-readable documentation.

## Workflow
1. **skills-architect**: Define the `SKILLS.md` format and core skill names.
2. **system-writer**: Author the `SKILL.md` content for the 3 core skills.
3. **wordpress-integrator**: Update `sync_skills_to_wp.py` with category support.
4. **test-engineer**: Run `sync-skills` and verify all results across the repo and WordPress.
