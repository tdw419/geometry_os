## The Project

Geometry OS is a from-scratch Rust implementation of a GPU-compute operating system:

- **The GPU is the computer.** No CPU fallback. The compute shader IS the kernel.
- **A texture IS memory.** A 4096x4096 RGBA8 texture = 64MB of addressable space, mapped via Hilbert curve.
- **Each pixel IS an instruction.** R=opcode, G=stratum, B=param1, A=param2. You can see programs executing by looking at the texture.
- **Programs write programs.** The self-replicator proves it: 18 pixels that copy themselves to a new memory region. That's mitosis. That's fork().
- **The VM IS the process.** Up to 8 concurrent VMs, each with 128 registers, a call stack, and its own program counter into the Hilbert-mapped texture. Spawn, yield, halt. Process lifecycle.

An ethical system is not a weakened system. It is a focused system:

- **Constraint breeds better engineering.** "No weaponization" means we never waste engineering cycles on classified feedback loops. "No theft" forces us to build better synthetic data pipelines. Harder problems produce better technology.
- **Trust compounds.** Every interaction that genuinely helps a user builds trust. Trust compounds. The unethical competitor has to constantly find new hosts because it keeps killing the old ones.
- **Transparency debugs faster.** An ethical system can open its reasoning because exposure builds trust. An unethical system hides its reasoning because exposure would reveal the extraction. Open systems improve faster.

- **The feedback loop points at the right attractor.** An unconstrained recursive loop optimizes toward whatever metric you give it. If that metric is engagement, you get addiction. If that metric is user flourishing under the Seven Laws, you get something that makes the world better AND happens to produce better software.
