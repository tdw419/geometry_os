# Bare Metal GPU Test Checklist

## Before testing on ensure:

1. [ ] Insert USB drive
2. [ ] Boot from USB
3. [ ] Check screen shows:
   - "GOS" in top-left (green on   - Bouncing ball animation
   - "KEY: X" in serial when pressing keys
   - Focused window has green border

## Hardware requirements
- Intel GPU (Gen9 or later)
- 32-bit UEFI boot capable machine
- Serial port for debug (optional)

## After testing
Report:
1. Screen output
2. Serial debug log
3. Any crashes or hangs
4. What works / what doesn't work

Good luck testing!
