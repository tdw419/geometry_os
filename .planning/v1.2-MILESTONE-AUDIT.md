## Integration Gaps

### Critical: DeltaHTTPHandler Not Wired

**Location:** `systems/pixel_compiler/serve/server.py` lines 477-482

**Problem:** DeltaServer is created but DeltaHTTPHandler is never registered with HTTPBootServer.

```python
# What exists (server.py:477-482):
self._delta_server = DeltaServer(...)
self.status.delta_enabled = True

# What's missing:
# self._http_server.register_handler(DeltaHTTPHandler(self._delta_server))
```

**Impact:**
- `/delta/list` returns 404 (looked up as file, not dynamic route)
- `/delta/<file>.json` returns 404 (same reason)
- `pixelrts update` command fails when fetching manifest

**Fix Required:**

1. Add `register_handler()` method to HTTPBootServer
2. Wire DeltaHTTPHandler in PixelRTSServer.start()

```python
# In http_server.py
def register_handler(self, handler):
    """Register a custom path handler."""
    self._custom_handlers.append(handler)

# In server.py, after creating delta server:
if self._delta_server:
    from systems.pixel_compiler.serve.delta_server import DeltaHTTPHandler
    self._http_server.register_handler(DeltaHTTPHandler(self._delta_server))
```
