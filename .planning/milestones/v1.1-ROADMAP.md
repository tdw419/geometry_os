# Milestone v1.1: Visual Diff

**Status:** ✅ SHIPPED 2026-03-08
**Phases:** 5
**Total Plans:** 4

## Overview

Visual Diff enables users to compare two .rts.png files and see byte-level differences with Hilbert space visualization. The `pixelrts diff` command provides terminal output with Rich formatting, JSON export, and RGBA channel breakdown.

## Phases

### Phase 5: Visual Diff Engine

**Goal**: Users can compare two .rts.png files and see visual diff of changes
**Depends on**: Phase 1 (existing PixelRTS v2 encoding)

## Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Reuse PixelRTSDecoder | No new PNG parsing code | ✓ Good |
| scipy.ndimage.label for regions | Connected component detection | ✓ Good |
| Rich output with fallback | Graceful degradation | ✓ Good |
| RGBA channel mapping via (byte_idx % 4) | Simple channel extraction | ✓ Good |
| Output limited to 20 regions | Prevent terminal flood | ✓ Good |
| Exit codes for scripting | Shell integration | ✓ Good |

## CLI Command Shipped

- `pixelrts diff <old.png> <new.png>` — Visual byte-level comparison

**Options:**
- `--json` — JSON output for scripting
- Exit codes: 0=identical, 1=different, 2=missing, 3=invalid
