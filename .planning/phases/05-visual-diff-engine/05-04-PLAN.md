Add `cmd_diff(args)` function to pixelrts_cli.py following existing patterns.

Implementation:
1. Import PixelRTSDiffer and format_diff_output from pixelrts_diff
2. Create PixelRTSDiffer instance
3. Call differ.diff(args.old, args.new)
4. If args.json: print json.dumps(result, indent=2, default=str)
5. Else: print(format_diff_output(result))
6. Return 0 if change_percent == 0 else 1 (exit code for scripting)

Handle errors:
- FileNotFoundError: Print error message, return 2
- Invalid PNG: Print error message, return 3
- General exception: Print error, return 1

Add subparser in main():
```python
diff_parser = subparsers.add_parser('diff', help='Compare two .rts.png files')
diff_parser.add_argument('old', help='Original .rts.png file')
diff_parser.add_argument('new', help='New .rts.png file')
diff_parser.add_argument('--json', action='store_true',
                        help='Output as JSON')
diff_parser.set_defaults(func=cmd_diff)
```
