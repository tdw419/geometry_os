  <action>
Add `_compute_channel_stats(self, diff_mask: np.ndarray, old_arr: np.ndarray, new_arr: np.ndarray) -> dict` method.

Implementation:
1. Initialize stats dict with channels: 'R', 'G', 'B', 'A'
2. For each channel (index 0-3):
   - Get mask for bytes at this channel: `channel_mask = (np.arange(len(diff_mask)) % 4) == channel_idx`
   - Combine with diff_mask: `changed_in_channel = diff_mask & channel_mask`
   - Count changed bytes in this channel
   - Calculate mean absolute change for changed bytes
3. Return dict with:
   - per_channel: dict mapping channel name to {changed: int, mean_delta: float}
   - most_changed_channel: channel name with most changes
   - least_changed_channel: channel name with fewest changes

Channel index mapping: R=0, G=1, B=2, A=3
</action>

print('Channel stats:', stats)
