  <action>
Create new file `systems/pixel_compiler/pixelrts_diff.py` with PixelRTSDiffer class.

Implementation requirements:
1. Import PixelRTSDecoder from pixelrts_v2_core
2. Import numpy for array operations
3. Create PixelRTSDiffer class with:
   - `__init__(self)`: Initialize decoder instance
   - `diff(old_path: str, new_path: str) -> dict`: Main comparison method

The diff() method must:
1. Read both PNG files as bytes
2. Use PixelRTSDecoder to decode each to raw bytes
3. Convert to numpy uint8 arrays
4. Pad shorter array to match lengths (np.pad with zeros)
5. Calculate three masks:
   - added: (old == 0) & (new != 0)
   - removed: (old != 0) & (new == 0)
   - changed: (old != new) & (old != 0) & (new != 0)
6. Return dict with:
   - old_file, new_file (paths)
   - added_bytes, removed_bytes, changed_bytes (int counts)
   - unchanged_bytes (total - added - removed - changed)
   - total_bytes (max length)
   - change_percent (rounded to 2 decimals)
   - diff_mask (boolean numpy array for downstream use)
   - old_metadata, new_metadata (from decoder.get_metadata())

Handle edge cases:
- Reset decoder._metadata = None between file decodes
- Handle empty files (return zeros)
- Extract grid_size from metadata (default 256)

DO NOT implement region grouping - that's plan 05-02.
DO NOT implement CLI output - that's plan 05-04.
