<objective>
Fix CommittedFileBooter to extract kernel/initrd binary data from committed files using offset metadata, and trim qcow2 extraction to disk_size only.

Purpose: The SnapshotExporter correctly stores kernel/initrd binary data appended to qcow2 data with offset metadata. However, CommittedFileBooter was written expecting filesystem paths instead of binary extraction. This plan fixes the extraction logic to properly extract each component.

Output: Working boot from committed files with preserved kernel/initrd.

Root Cause Analysis:
1. extract_qcow2() writes ALL decoded data (qcow2 + kernel + initrd) to qcow2 file - corrupts disk
2. boot() looks for metadata.original_kernel/initrd (paths) but design stores binary data at offsets
3. No code exists to extract kernel/initrd bytes from combined data using offsets
4. Decoder instance and metadata are not stored after decode, losing offset information

The Fix:
1. Store decoder instance after decode to access metadata with offsets
2. In extract_qcow2(): trim decoded data to disk_size bytes before writing
3. Add _extract_kernel() and _extract_initrd() methods to extract bytes using offsets
4. Write extracted kernel/initrd to temp files
5. Pass temp file paths to QemuBoot.boot()
</objective>

```python
def _extract_kernel(self, output_dir: Path) -> Optional[Path]:
    """
    Extract kernel from committed file's combined data.

    Uses offset metadata to locate kernel bytes in the decoded data.

    Args:
        output_dir: Directory to write extracted kernel

    Returns:
        Path to extracted kernel file, or None if not present
    """
    if not self._decoder:
        logger.warning("No decoder available - call extract_qcow2 first")
        return None

    metadata = self._decoder.get_metadata()
    if not metadata:
        logger.warning("No metadata available")
        return None

    # Check if kernel is present
    if not metadata.get('has_kernel', False):
        logger.info("No kernel stored in committed file")
        return None

    offsets = metadata.get('offsets', {})
    kernel_info = offsets.get('kernel')
    if not kernel_info:
        logger.warning("Kernel flag set but no offset info")
        return None

    # Re-decode to get combined data (or store it in extract_qcow2)
    # Actually, we need to store combined_data - let's add self._combined_data
    # For now, re-decode (less efficient but works)
    with open(self.rts_png_path, 'rb') as f:
        png_data = f.read()

    combined_data = self._decoder.decode(png_data)

    # Extract kernel bytes
    start = kernel_info['offset']
    size = kernel_info['size']
    kernel_data = combined_data[start:start + size]

    # Verify hash if present
    expected_hash = kernel_info.get('sha256')
    if expected_hash:
        import hashlib
        actual_hash = hashlib.sha256(kernel_data).hexdigest()
        if actual_hash != expected_hash:
            logger.warning(f"Kernel hash mismatch: expected {expected_hash[:16]}..., got {actual_hash[:16]}...")
            return None

    # Write to temp file
    kernel_path = output_dir / "kernel"
    with open(kernel_path, 'wb') as f:
        f.write(kernel_data)

    logger.info(f"Extracted kernel to {kernel_path} ({size} bytes)")
    return kernel_path


def _extract_initrd(self, output_dir: Path) -> Optional[Path]:
    """
    Extract initrd from committed file's combined data.

    Uses offset metadata to locate initrd bytes in the decoded data.

    Args:
        output_dir: Directory to write extracted initrd

    Returns:
        Path to extracted initrd file, or None if not present
    """
    if not self._decoder:
        logger.warning("No decoder available - call extract_qcow2 first")
        return None

    metadata = self._decoder.get_metadata()
    if not metadata:
        logger.warning("No metadata available")
        return None

    # Check if initrd is present
    if not metadata.get('has_initrd', False):
        logger.info("No initrd stored in committed file")
        return None

    offsets = metadata.get('offsets', {})
    initrd_info = offsets.get('initrd')
    if not initrd_info:
        logger.warning("Initrd flag set but no offset info")
        return None

    # Re-decode to get combined data
    with open(self.rts_png_path, 'rb') as f:
        png_data = f.read()

    combined_data = self._decoder.decode(png_data)

    # Extract initrd bytes
    start = initrd_info['offset']
    size = initrd_info['size']
    initrd_data = combined_data[start:start + size]

    # Verify hash if present
    expected_hash = initrd_info.get('sha256')
    if expected_hash:
        import hashlib
        actual_hash = hashlib.sha256(initrd_data).hexdigest()
        if actual_hash != expected_hash:
            logger.warning(f"Initrd hash mismatch: expected {expected_hash[:16]}..., got {actual_hash[:16]}...")
            return None

    # Write to temp file
    initrd_path = output_dir / "initrd"
    with open(initrd_path, 'wb') as f:
        f.write(initrd_data)

    logger.info(f"Extracted initrd to {initrd_path} ({size} bytes)")
    return initrd_path
```

Note: These methods re-decode the PNG each time. For optimization, you could store `self._combined_data` in extract_qcow2 and reuse it here, but re-decoding is simpler and ensures correctness.
