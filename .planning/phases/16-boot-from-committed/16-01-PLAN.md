<task type="auto">
  <name>Task 1: Create CommittedFileBooter class</name>
  <files>systems/pixel_compiler/boot/committed_boot.py</files>
  <action>
Create `CommittedFileBooter` class at `systems/pixel_compiler/boot/committed_boot.py`:

1. **ContainerType enum** - BOOTABLE, VM_SNAPSHOT
2. **CommittedFileMetadata dataclass** - stores extracted metadata (type, snapshot_tag, original_kernel, original_initrd, source_format, disk_size)
3. **CommittedFileBooter class** with:

   - `__init__(rts_png_path, memory="2G", cpus=2, vnc_display=0, verbose=False, network_mode=None, socket_config=None)` - similar to BootBridge signature
   - `detect_container_type(rts_png_path) -> ContainerType` - static method that reads PNG tEXt metadata to check for "type" field
   - `extract_metadata(rts_png_path) -> CommittedFileMetadata` - reads PNG metadata including original kernel/initrd offsets (if stored during commit)
   - `extract_qcow2(rts_png_path, output_dir) -> Path` - decodes PNG to qcow2 bytes, writes to temp file
   - `boot(cmdline=None, extra_qemu_args=None) -> BootResult` - main boot method that:
     a. Creates temp directory for extraction
     b. Extracts qcow2 disk to temp file
     c. If original kernel/initrd stored in metadata, extract them; otherwise use boot discovery from qcow2
     d. Starts QEMU with `-kernel`, `-initrd`, and `-drive file=qcow2` (for disk changes persistence)
     e. Returns BootResult with process, vnc_port, etc.
   - `stop()` - cleanup QEMU and temp files
   - Context manager support (`__enter__`, `__exit__`)

4. **Key design decisions**:
   - Use `tempfile.TemporaryDirectory` for extraction with cleanup in `stop()` and `__exit__`
   - Store boot_bridge-style configuration for QEMU (memory, cpus, vnc, network)
   - Reuse `QemuBoot` class for actual QEMU process management
   - For vm-snapshot files, the qcow2 IS the disk, so boot with `-drive file=qcow2,snapshot=off` to persist changes
   - If original kernel/initrd not in metadata, attempt to extract from qcow2 using `guestfish` or similar (log warning, may fail)

5. **Integration with existing code**:
   - Import `PixelRTSDecoder` from `pixelrts_v2_core` for decoding PNG to qcow2 bytes
   - Import `QemuBoot`, `QemuConfig`, `VMStatus` from `integration.qemu_boot`
   - Use `BootResult` from `boot_bridge.py` (copy the dataclass if not importable)
   - Follow same error handling patterns as BootBridge

Do NOT modify BootBridge or other existing classes. This is a NEW class for a NEW boot path.
