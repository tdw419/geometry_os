Create a new file `systems/pixel_compiler/boot/vm_snapshot.py` with:

1. **SnapshotError exception:**
   - Custom exception for snapshot operation failures

2. **SnapshotState enum (reuse pattern from infinite_map_snapshot_rollback.py):**
   - CREATING, COMPLETE, PARTIAL, CORRUPTED, DELETING

3. **VMSnapshotMetadata dataclass:**
   - snapshot_id: str (unique identifier)
   - tag: str (QEMU snapshot tag)
   - container_name: str
   - created_at: datetime
   - state: SnapshotState
   - vm_memory: str
   - description: Optional[str] = None
   - size_bytes: int = 0
   - to_dict() method for serialization
   - from_dict() classmethod for deserialization

4. **SnapshotResult dataclass:**
   - success: bool
   - tag: str
   - metadata: Optional[VMSnapshotMetadata] = None
   - error_message: Optional[str] = None

5. **SnapshotInfo dataclass (parsed from 'info snapshots'):**
   - id: int
   - tag: str
   - size: str
   - date: str
   - vm_clock: str

6. **VMSnapshotManager class:**
   - `__init__(self, qemu_boot, container_name: str)` - Store QemuBoot reference
   - `create_snapshot(self, tag: str, description: str = "", timeout: float = 30.0) -> SnapshotResult`
     - Validate VM is running (check qemu_boot._process and _monitor_socket)
     - Validate tag format (alphanumeric, dash, underscore only)
     - Calculate timeout based on memory size (2GB = 5s baseline, scale linearly)
     - Send `savevm {tag}` via send_monitor_command()
     - Create and return VMSnapshotMetadata
   - `list_snapshots(self) -> List[SnapshotInfo]`
     - Send `info snapshots` command
     - Parse tabular output format (ID, TAG, VM SIZE, DATE, VM CLOCK)
     - Return list of SnapshotInfo objects
   - `restore_snapshot(self, tag: str) -> SnapshotResult`
     - Verify snapshot exists via list_snapshots()
     - Send `loadvm {tag}` command
     - Return SnapshotResult
   - `delete_snapshot(self, tag: str) -> SnapshotResult`
     - Send `delvm {tag}` command
     - Return SnapshotResult

Key implementation details:
- Use existing QemuBoot.send_monitor_command() - DO NOT create new socket code
- Tag validation: `re.match(r'^[\w-]+$', tag)` - raise SnapshotError if invalid
- Parse 'info snapshots' output with regex: `^\s*(\d+)\s+(\S+)\s+([\d.]+\s+\w+)\s+(\d{4}-\d{2}-\d{2}\s+\d{2}:\d{2}:\d{2})\s+([\d:]+)`
