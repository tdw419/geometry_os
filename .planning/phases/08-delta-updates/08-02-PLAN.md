<task type="auto">
  <name>Task 5: Add pixelrts patch CLI command</name>
  <files>systems/pixel_compiler/pixelrts_cli.py</files>
  <action>
Add `pixelrts patch` command to CLI:

1. Add cmd_patch(args) function:
   ```python
   def cmd_patch(args):
       """Handle delta patch application command."""
       from systems.pixel_compiler.delta_patch import apply_delta_patch, PatchError

       base_path = args.base
       manifest_path = args.manifest
       output_path = args.output

       try:
           result_path = apply_delta_patch(
               base_path=base_path,
               manifest_path=manifest_path,
               output_path=output_path,
               validate_checksums=not args.skip_validation
           )
           print(f"Patched file written to: {result_path}")
       except PatchError as e:
           print(f"Error: {e.message}", file=sys.stderr)
           if e.region_index is not None:
               print(f"  Failed at region {e.region_index}", file=sys.stderr)
           sys.exit(1)
   ```

2. Add subparser in main():
   ```python
   # Patch command
   patch_parser = subparsers.add_parser('patch', help='Apply delta patch to .rts.png file')
   patch_parser.add_argument('base', help='Path to base .rts.png file')
   patch_parser.add_argument('manifest', help='Path to delta manifest JSON')
   patch_parser.add_argument('-o', '--output', help='Output file (default: overwrite base)')
   patch_parser.add_argument('--skip-validation', action='store_true',
                             help='Skip checksum validation (dangerous)')
   patch_parser.set_defaults(func=cmd_patch)
   ```

3. Add success/failure exit codes
