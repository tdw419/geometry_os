Add generate_delta_manifest() function that leverages PixelRTSDiffer:

1. Import PixelRTSDiffer from pixelrts_diff

2. Create function:
   ```python
   def generate_delta_manifest(old_path: str, new_path: str) -> DeltaManifest:
       """
       Generate a delta manifest between two .rts.png files.

       Uses PixelRTSDiffer to identify changed regions, then computes
       checksums and builds the manifest structure.

       Args:
           old_path: Path to the original .rts.png file
           new_path: Path to the new/modified .rts.png file

       Returns:
           DeltaManifest with all changed regions and metadata
       """
   ```

3. Implementation steps:
   a. Use PixelRTSDiffer().diff(old_path, new_path) to get diff result
   b. Compute SHA-256 checksums for old and new files (entire file)
   c. Extract diff_mask from result
   d. Find contiguous changed regions from diff_mask
   e. For each region:
      - Calculate offset and length
      - Compute SHA-256 of bytes in that region (both old and new)
      - Create DeltaRegion object
   f. Build DeltaManifest with all metadata

4. Helper function _find_contiguous_regions(diff_mask, min_size=64):
   - Convert boolean diff_mask to list of (start, end) tuples
   - Merge small gaps (< min_size) to reduce region count
   - Return list of (offset, length) tuples

5. Helper function _sha256_region(data: bytes, offset: int, length: int) -> str:
   - Extract bytes[offset:offset+length]
   - Return hex digest of SHA-256 hash
</action>

<task type="auto">
  <name>Task 4: Add pixelrts delta CLI command</name>
  <files>systems/pixel_compiler/pixelrts_cli.py</files>
  <action>
Add `pixelrts delta` command to CLI:

1. Add cmd_delta(args) function:
   ```python
   def cmd_delta(args):
       """Handle delta manifest generation command."""
       from systems.pixel_compiler.delta_manifest import generate_delta_manifest, DeltaManifest

       old_path = args.old
       new_path = args.new
       output_path = args.output

       # Generate manifest
       manifest = generate_delta_manifest(old_path, new_path)

       # Output to file or stdout
       if output_path:
           manifest.save(output_path)
           print(f"Delta manifest saved to: {output_path}")
       else:
           print(manifest.to_json())

       # Print summary
       if not args.quiet:
           print(f"\nSummary:")
           print(f"  Regions changed: {len(manifest.regions)}")
           print(f"  Bytes changed: {manifest.new_size - manifest.unchanged_bytes:,}")
           print(f"  Change ratio: {manifest.compression_ratio:.1f}%")
   ```

2. Add subparser in main():
   ```python
   # Delta command
   delta_parser = subparsers.add_parser('delta', help='Generate delta manifest between two versions')
   delta_parser.add_argument('old', help='Path to old .rts.png file')
   delta_parser.add_argument('new', help='Path to new .rts.png file')
   delta_parser.add_argument('-o', '--output', help='Output manifest file (default: stdout)')
   delta_parser.add_argument('-q', '--quiet', action='store_true', help='Suppress summary output')
   delta_parser.set_defaults(func=cmd_delta)
   ```

3. Handle errors gracefully (file not found, invalid files)
