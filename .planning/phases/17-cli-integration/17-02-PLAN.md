  <how-to-verify>
## End-to-End Verification Steps

1. **Boot a container:**
   ```bash
   pixelrts boot some-container.rts.png --background
   ```

2. **List running containers:**
   ```bash
   pixelrts ps
   ```
   Note the container name.

3. **Commit the container:**
   ```bash
   pixelrts commit <container-name> committed.rts.png
   ```
   Expected: Progress output showing stages, success message with file size.

4. **Verify the committed file exists:**
   ```bash
   ls -la committed.rts.png
   ```

5. **Boot the committed file:**
   ```bash
   pixelrts boot committed.rts.png
   ```
   Expected: Auto-detects vm-snapshot type, boots successfully.

6. **Test --no-verify flag:**
   ```bash
   pixelrts commit <container-name> fast-commit.rts.png --no-verify
   ```
   Expected: Faster commit without verification step.

7. **Test --snapshot flag (if snapshots exist):**
   ```bash
   pixelrts snapshot create <container-name> --tag test-snap
   pixelrts commit <container-name> from-snap.rts.png --snapshot test-snap
   ```

8. **Run unit tests:**
   ```bash
   python3 -m pytest tests/unit/test_pixelrts_cli_commit.py -v
   ```
  </how-to-verify>
