2. Validate inputs:
   - Check container name is provided
   - Check output path is provided and ends with .rts.png
   - Look up container in MultiBootManager

3. Get BootBridge from running container:
```python
manager = MultiBootManager()
container = manager.get_container(args.container)
if not container:
    print(f"Error: Container '{args.container}' not found", file=sys.stderr)
    print("Use 'pixelrts ps' to list running containers", file=sys.stderr)
    return 1
boot_bridge = container.get('boot_bridge')
if not boot_bridge:
    print(f"Error: Container '{args.container}' has no boot_bridge", file=sys.stderr)
    return 1
```

4. Create SnapshotExporter and run export:
```python
exporter = SnapshotExporter(boot_bridge)
result = exporter.export(
    output_path=Path(args.output),
    tag=args.snapshot,
    timeout=args.timeout,
    verify=not args.no_verify
)
```

5. Handle result:
   - On success: print output path and size
   - On failure: print error message and return 1

6. Progress output (if not quiet):
   - Print stage updates during export

Place the function near cmd_snapshot_create (around line 1636) to keep related commands together.
