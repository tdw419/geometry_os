  <action>
    Create `systems/pixel_compiler/boot/ephemeral_boot.py` with EphemeralBooter class:

    1. **EphemeralBootError exception class** - raised when ephemeral boot fails

    2. **EphemeralBooter class**:
       - `__init__(rts_png_path, memory="2G", cpus=2, vnc_display=0, verbose=False, network_mode=None, socket_config=None)`
       - Creates temp directory via `tempfile.mkdtemp(prefix="pixelrts-ephemeral-")`
       - Copies source .rts.png to temp location: `{temp_dir}/{original_name}`
       - Stores original path and temp path for later reference
       - Detects container type (bootable vs vm-snapshot) via CommittedFileBooter.detect_container_type
       - Creates appropriate inner booter (BootBridge for bootable, CommittedFileBooter for vm-snapshot) pointing to TEMP copy
       - Sets `self.is_ephemeral = True`

    3. **boot() method**:
       - Delegates to inner booter's boot() method
       - Returns BootResult from inner booter
       - If boot fails, cleans up temp directory immediately

    4. **stop() method**:
       - Calls inner booter's stop()
       - THEN removes temp directory via `shutil.rmtree(self._temp_dir)`
       - Safe to call multiple times

    5. **Context manager support** (`__enter__`, `__exit__`):
       - Ensures cleanup on context exit

    6. **get_status() method**:
       - Returns dict with ephemeral=True and inner booter status

    7. **original_path property**:
       - Returns Path to original source file (for commit operations)

    Pattern after CommittedFileBooter's structure (temp file management, context manager, delegation).

    DO NOT modify original source file at any point.
  </action>
  <verify>
    python3 -c "from systems.pixel_compiler.boot.ephemeral_boot import EphemeralBooter, EphemeralBootError; print('Import OK')"
  </verify>
  <done>
