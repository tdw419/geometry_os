    - EphemeralBooter class for temp file management

    - [E] indicator in ps output
    - Crash cleanup via atexit and signal handlers
    - Commit support for ephemeral containers
    - Multi-boot ephemeral support
  </what-built>
  <how-to-verify>
    1. Run unit tests: `python3 -m pytest tests/unit/test_ephemeral_boot.py tests/unit/test_multi_boot_manager.py -v --tb=short`

    2. Test single ephemeral boot:
       ```
       pixelrts boot --ephemeral test.rts.png
       # Make changes inside container
       # Exit container
       # Verify test.rts.png unchanged (md5sum before/after)
       ```

    3. Test ps indicator:
       ```
       pixelrts boot --ephemeral test.rts.png &
       pixelrts ps
       # Should show [E] indicator
       ```

    4. Test commit from ephemeral:
       ```
       pixelrts boot --ephemeral test.rts.png &
       pixelrts commit test saved.rts.png
       # Verify saved.rts.png contains changes
       # Verify test.rts.png unchanged
       ```

    5. Test multi-boot ephemeral:
       ```
       pixelrts boot --ephemeral a.rts.png b.rts.png
       # Both should be ephemeral
       ```

    6. Verify crash cleanup:
       ```
       pixelrts boot --ephemeral test.rts.png &
       kill -9 <pid>
       # Verify no orphan temp directories in /tmp/pixelrts-ephemeral-*
       ```
  </how-to-verify>
