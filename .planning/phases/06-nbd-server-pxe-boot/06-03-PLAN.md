ServeProgress displays:
- Server configuration panel (interface, IP, ports)
- Active clients table with columns: IP, MAC, Stage, Progress, Status
- Transfer statistics (total bytes, current rate)
- Log messages panel for errors/info

ClientTracker tracks per-client boot stages:
- DHCP - Client sent DHCPDISCOVER
- TFTP - Client downloading boot files
- NBD - Client connected to NBD, loading kernel
- KERNEL - Kernel booting
- COMPLETE - Boot successful
- FAILED - Boot failed with error

Rich components to use:
- `Console` for output
- `Table` for client list
- `Panel` for server config
- `Progress` for transfer bars
- `Live` for real-time updates

Key methods:

- `update_config(interface, ip, ports)` - Show server config
- `add_client(ip, mac)` - Add new client to tracker
- `update_client_stage(ip, stage, progress)` - Update client status
- `remove_client(ip)` - Remove client (keep in history)
- `log_message(msg, level)` - Add log entry
- `stop_display()` - Clean up display
