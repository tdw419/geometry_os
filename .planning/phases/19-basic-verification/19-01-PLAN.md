  <name>Task 2: Implement StructureVerifier for PNG validation</name>
  <files>
    systems/pixel_compiler/verification/structure_verifier.py
  </files>
  <action>
    Create StructureVerifier that validates PNG structure and PixelRTS metadata:

    **systems/pixel_compiler/verification/structure_verifier.py**

    StructureVerifier class (extends VerificationStep):
    - name = "structure"
    - description = "Validates PNG structure and PixelRTS metadata"

    verify() method performs these checks:

    1. **PNG Signature Check**
       - Verify file starts with PNG signature: 89 50 4E 47 0D 0A 1A 0A
       - FAIL if signature missing/invalid with message "Invalid PNG signature"

    2. **PNG Chunk Structure Check**
       - Use PIL to open image
       - FAIL if PIL cannot open with specific error message
       - Verify image is square (width == height)
       - FAIL if not square with dimensions in message

    3. **Grid Size Validation**
       - Verify grid_size is power of 2
       - FAIL if not power of 2 with actual grid_size in message

    4. **PixelRTS Metadata Check**
       - Look for tEXt chunk containing "PixelRTS" prefix
       - Use PixelRTSMetadata.decode_png_text() to parse
       - FAIL if no PixelRTS metadata found
       - FAIL if metadata parsing fails with specific error

    Each check creates a StepResult. The overall result is FAIL if any check fails.

    Reference: PixelRTSMetadata.decode_png_text() in pixelrts_v2_core.py
  </action>
